fitSharp
Release22
Features
  • Configure Fixture can issue multiple commands in one table.
  • Configure Fixture has a new 'processor' facility.
  • The Find Member Pattern Operator finds members with a 'MemberPattern' attribute containing a regular expression that matches the contents of a member cell.
  • Test output uses Syntax Coloring to indicate how each cell was processed..
Bugs Fixed
  • Graceful naming appends 'fixture' to a class name only in places where a fixture name is expected.
  • A fixture can override the standard Flow Fixture Keywords.
