fitSharp
Release20
This release contains some significant changes to the internal architecture of fitSharp that will enable new features in future releases. Some obsolete features have been removed or deprecated:
  • Test Runner has been removed.
  • The static methods in the Fixture class to access Symbol Values have been marked obsolete.

Features
  • Cell Differences for strings are described.
  • App Domain setup information can be specified in a Suite Configuration File.
  • Optional assemblies to be loaded can be specified in a Suite Configuration File.
  • Folder Runner Dry Run option lists test files without executing tests.
  • Test files can be selected for execution by using Folder Runner With Tags.
  • DbFit supports MySql databases.
  • DbFit supports Sybase databases.

Bugs Fixed
  • A relative App Config File path is loaded relative to the application base path.
  • DbFit handles multi-byte Unicode characters correctly for SqlServer.
  • Slim array parsing is handled correctly.
