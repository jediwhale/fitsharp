fitSharp
Release18
Features
  • Additional SQLServer data types are supported in DbFit.
  • DbFit has a CommandTimeOut option.
  • An App Config File can be specified in the Suite Configuration File.
  • An App Config File can be specified with a relative path.
Bugs Fixed
  • DbFit can be used in standalone mode.
  • A cell with "null" matches a DBNull item in a DataRow.
  • A cell with "null" is parsed correctly for array types.
  • The default Do Fixture doesn't require the fitlibrary namespace to be specified.
  • The Column Fixture Target Object can be changed as the fixture code executes.
  • The ApartmentState in the Suite Configuration File can be specified for Slim tests.
  • A member name can be quoted with a trailing question mark.
  • The symbol recall operator can be used in collection fixture cells without being evaluated multiple times.
