fitSharp
Release21
Features
  • Folder Runner Set Up Tear Down pages are no longer copied to the output directory.
  • Folder Runner copies the stylesheet to the output directory before any result files are written.
  • Include Fixture include other pages within a story test page. This is a preliminary version that only works with Folder Runner. Additional functions will be available in the next release.
  • Configure Fixture can set, display and clear Symbol Values.
  • Method Execution handles generic methods.
  • Debug Fixture launches the debugger.
  • Error reporting in Array Fixture and related fixtures has been improved.
  • A new Flow Fixture Keyword check fields for <method> <parms> wraps a result so it can be checked with Array Fixture.
Bugs Fixed
  • Symbol Values can hold arrays.
  • Slim array parsing is handled correctly.
  • Symbol Values in Fixture Arguments are handled correctly.
