fitSharp
Release24
Features
  • Alternate assembly folders can be specified in the command line for Loading Assemblies.
  • Literal Values can specify a constant value wherever Method Execution is allowed.
  • The Match Value Fixture obtains a value and executes rows that begin with a matching value.
  • DbFit can use connection strings from the application configuration file..
  • The Parse Array operator can parse multiple Symbol Values.
  • Slim uses the culture specfied in the Suite Configuration File.
  • Slim protocol v0.4 is supported.
Bugs Fixed
  • DbFit SQL Server times use the .NET TimeSpan type.
  • A Symbol Value containing a string is parsed as the correct type for a Method Execution parameter.
  • Slim fixture chaining works correctly.
