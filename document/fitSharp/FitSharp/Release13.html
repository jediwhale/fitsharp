Features
  • A Slim test can be stopped by throwing an exception containing 'StopTest' in the exception class name.
  • The Abandon Story Test exception can have a custom message.
  • The Parse Date Operator allows relative dates.
  • Files in the assembly path with a .jar extension are ignored.

Bugs Fixed
  • Make tests insensitive to new line characters.
  • Slim lists are recognized only when count prefix is present.
