fitSharp
Release15
Features
  • Enumerations and nullable types added to Slim Type Conversions.
  • Dictionaries added to Slim Type Conversions to support FitNesse 'hash widget'.
  • Suite set up and suite tear down files are executed by Folder Runner.
  • Fit Library named objects and FitNesse symbols have been merged.
  • Fit Version Fixture is integrated with the Suite Configuration File.
  • FitLibrary tree classes are serializable.
  • Slim library instances added to support Slim protocol V0.1.
Bugs Fixed
  • Constructor on the first fixture on a page is called only once.
  • A member name beginning with 'new' is not treated as a 'new' keyword by Column Fixture.
  • Empty Tree Values match a blank cell.
