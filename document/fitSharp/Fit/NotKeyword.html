Not Keyword
The Not Keyword is one of the Flow Fixture Keywords. It expects a boolean false or an exception.

when input is'sample do'
set 'boolean field' true
not 'boolean field'
set 'boolean field' false
not 'boolean field'
not throw oops		then output is
sample do
set boolean field true
not boolean field
set boolean field false
not boolean field
not throw oops

The Reject Keyword is a synonym.

when input is'sample do'
set 'boolean field' true
reject 'boolean field'
set 'boolean field' false
reject 'boolean field'
reject throw oops		then output is
sample do
set boolean field true
reject boolean field
set boolean field false
reject boolean field
reject throw oops
