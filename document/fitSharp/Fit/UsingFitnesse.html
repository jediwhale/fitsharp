Using Fitnesse
To run tests from the FitNesse wiki, add the following to the root page of your Fit tests. This could be the root of your FitNesse wiki (e.g., localhost:8080/root) or on the Fit test page or suite page.
!path c:\myfolder\mytest.dll
!define COMMAND_PATTERN {%m -r fitnesse.fitserver.FitServer,c:\program files\fitsharp\fit.dll %p}
!define TEST_RUNNER {c:\program files\fitsharp\Runner.exe}

To run FitNesse tests from the command line, you can use the following.
c:\program files\fitsharp\runner -r fitnesse.fitserver.TestRunner,c:\program files\fitsharp\fit.dll
    host port TestPage c:\myfolder\mytest.dll
