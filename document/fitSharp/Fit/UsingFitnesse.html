Using Fitnesse
We can store, edit, and run tests using the FitNesse framework.

There is a choice of two formats for writing tests:
  • Html Table Format
  • Plain Text Format
We can mix tests in either format within a test suite, but an individual test page must be in one format.

The FitNesse environment must be customized to Run Tests With FitNesse.
