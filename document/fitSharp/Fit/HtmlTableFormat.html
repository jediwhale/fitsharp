Html Table Format
This is the classic format that you will see described in the Fit literature. We write a test page directly in HTML, or with wiki markup that translates to HTML. fitSharp processes the tables in the test page and ignores the rest of the page. So we can include additional information anywhere else on the page. Html Comments can be included. Tidy Html is handled.
