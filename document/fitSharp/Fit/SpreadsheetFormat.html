fitSharp
Spreadsheet Format
This is an early release feature and will be enhanced in future releases.

fitSharp can run tests from spreadsheet files. Files with the extension .xlsx are assumed to be spreadsheets. Only the first sheet in a file is processed. Test tables are separated with one or more blank rows. Only cell values are used. Any cell formatting is ignored.
