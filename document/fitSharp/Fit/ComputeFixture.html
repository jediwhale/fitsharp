Compute Fixture
The Compute Fixture checks the results of a function.

The second row of a Compute Fixture table is a header row: the last cell is the name of a function and the other cells are the names of the function parameters. The subsequent rows contain input values for the parameters and the expected value of the function's result. The Compute Fixture uses the current System Under Test to perform the requested function.
