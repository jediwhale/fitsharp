Parse Date
The Parse Date operator lets you include relative date expressions in input or expected value cells for DateTime values. The relative date can be "today", "today+n" or "today-n" where n is a number of days.

when input is'sample domain'
set datetimefield today
'add days' 2
check datetimefield today+2		then output is
sample domain
set datetimefield today
add days 2
check datetimefield today+2
