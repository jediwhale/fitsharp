Cell Handlers
An input value cell or an expected value cell is normally handled by parsing the cell contents based on the value type. Special processing of cell contents is implemented by a set of classes known as cell handlers. There is a variety of cell handlers provided (*list them*), and you can write your own (*show example*).

The cell handlers currently active are defined in Cell Handler Lists.

These are the standard cell handlers for the core Fit fixtures:
  • Exception Keyword Handler
  • Fail Keyword Handler
  • Numeric Comparison Operators
  • Symbol Values
...more...

These are some additional cell handlers provided that you can add to a list if you need them:
  • Floating Point Handler
  • Guid Handler
  • Reg Ex Handler
  • String Handler
