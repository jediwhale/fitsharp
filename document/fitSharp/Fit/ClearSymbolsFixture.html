Clear Symbols Fixture

The ClearSymbols fixture erases all Symbol Values.

when input is'sample domain'

set name Bob
check name >>mysymbol
check name <<mysymbol

'clear symbols'

set name null
check name <<mysymbol		then output is
sample domain

set name Bob
check name >>mysymbol Bob
check name <<mysymbol Bob

clear symbols

set name null
check name <<mysymbol null
