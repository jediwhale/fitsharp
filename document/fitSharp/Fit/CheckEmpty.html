Check Empty
The Check Empty operator handles an empty expected value cell. The operator displays the actual value instead of comparing values.

when input is'sample domain'

set name Bob
check name ''

set name null
check name ''

set name blank
check name ''		then output is
sample domain

set name Bob
check name Bob

set name null
check name null

set name blank
check name blank
