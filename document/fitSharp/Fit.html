fitSharp
Fit
fitSharp includes a .NET implementation of the Framework for Integrated Test (Fit) tool, including a .NET version of FitLibrary. We can use it to test .NET applications with story tests written in a tabular format. The test table cells contain keywords, class and method names to be executed, input data, and expected results. There are two choices for working with Fit tests:
  • Using Fitnesse
  • Using The File System
The Feature List describes what fitSharp Fit can do.

Testing .NET applications requires Loading Assemblies and possibly specifying App Config Files.
