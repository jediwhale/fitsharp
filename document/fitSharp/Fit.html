fitSharp
Fit
fitSharp includes a .NET implementation of the Framework for Integrated Test (Fit) tool, including a .NET version of FitLibrary. You can use it to test .NET applications with story tests stored in the FitNesse wiki or in plain old HTML files.

fitSharp Fit is now available as a technology preview release at the github download page. This implementation is based on the Fitnesse.NET code base, but it has made some Breaking Changes to clean up historical inconsistencies and provide a platform for new features.
