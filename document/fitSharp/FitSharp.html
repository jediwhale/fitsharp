Fit Sharp
fitSharp is a set of open-source functional testing tools for .NET, inspired by the Framework for Integrated Test (Fit).

The latest fitSharp release can be downloaded from github.

These are the tools available here:
  • Fit: the Framework for Integrated Test, with runners for FitNesse and batch.
  • Slim: the Simple List Invocation Method for FitNesse.
  • DbFit: Fixtures for database set up and verification.
Here's what else is available on this site:
  • Release Information lists new features and bug fixes implemented in each release.
  • Useful Links will take you to more information about functional testing.
  • The index of story tests run for the latest release.
  • The Suite Set Up for the story tests.
